# Cancer_classification-MPOnline

Dataset Used: Brain Tumor Classification MRI Dataset by Sartaj Bhuvaji on Kaggle.

It contains 4 classes of brain MRI scans:
1. Glioma Tumor
2. Meningioma Tumor
3. No Tumor
4. Pituitary Tumor

You can find the dataset and all its details here: https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri

Results: the highest accuracy was 71% at epoch 09
